# NL-Portal App 3.0.4_3.0.3-1

Een release van de NL-Portal App bundelt twee onafhankelijk geversioneerde onderdelen. De releasetag volgt het formaat
`{backend}_{frontend}-{revisie}`; deze release bevat NL-Portal Backend Libraries 3.0.4 en NL-Portal Frontend Libraries
3.0.3.

## Nieuwe Functionaliteit

De volgende functionaliteiten zijn nieuw toegevoegd:

* NL-Portal libraries bijgewerkt naar Backend Libraries 3.0.4 en Frontend Libraries 3.0.3.
* Nieuwe omgevingsvariabele `NLPORTAL_CONFIG_FORM_PROPERTIES_FORMDEFINITIONOBJECTTYPEURL` toegevoegd voor het
  instellen van de objecttype-URL van formulierdefinities in de Objecten API. De backend gebruikt deze URL om te
  valideren dat een opgehaald object daadwerkelijk een formulierdefinitie is.
